# IT1214 Student Grade Management System

## Project Description

This is a Java Object-Oriented Programming (OOP) application developed to manage student records and grades.

## Features

* Add Student
* Display All Students
* Search Student by ID
* Calculate Average Marks
* Exit Program

## Classes Used

### Student Class

Stores student information:

* Student ID
* Student Name
* Marks

### StudentManagement Class

Handles:

* User menu
* Student record management
* Search functionality
* Average mark calculation

## Technologies Used

* Java
* ArrayList
* Scanner
* Git & GitHub

## How to Run

1. Compile the Java files:
   javac Student.java StudentManagement.java

2. Run the program:
   java StudentManagement
